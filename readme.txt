Embed the Counter.App booking form quick and easy!

== Description ==

The current plugin allows the Counter.app customers to embed the Counter booking forms on their WordPress websites quick and easy. The customers only need to install and activate the plugin and then go to wp-admin -> settings -> counter booking widget for finishing the configuration. And that's all...

== 3rd party (i.e. external) services used ==

All the information collected by the booking form in the website frontend is being sent to the Counter.app booking engine. We've developed this plugin to allow the Counter.app customers to embed the booking forms easier in their WordPress websites.
Details about Counter.app can be found here: https://counter.app
Additionally, here you can find their terms of service and privacy policy: https://counter.app/privacy-policy

== Frequently Asked Questions ==

= Is the plugin free? =

Yes!

= Where can I find the plugin administration page? =

It will be available after installing and activating the plugin here: wp-admin -> settings -> counter booking widget.
